# TechLens-AI 🚀
**Agentic RAG for Technical Architecture Analysis**

TechLens-AI is a professional-grade solution designed to analyze complex technical documentation and source code using Generative AI. It transforms unstructured data into actionable architectural insights and interactive diagrams.

## ✨ Key Features (Roadmap)
- **Agentic RAG:** Intelligent context retrieval using Vector Databases.
- **Auto-Architecture:** Generates Mermaid.js diagrams from code/docs analysis.
- **Cloud Native:** Fully containerized (Docker) and deployed on DigitalOcean.
- **Senior-Grade Code:** Built with FastAPI, Clean Architecture, and Pydantic.

## 🛠 Tech Stack
- **Backend:** Python, FastAPI, LangChain, OpenAI/Claude API.
- **Frontend:** React, Tailwind CSS, Mermaid.js.
- **Infrastructure:** Docker, DigitalOcean App Platform, Pinecone (Vector DB).
